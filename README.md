# 🏥 Hospital Management Website

## Project Overview
The Hospital Management Website is a responsive web application developed using HTML, CSS, and JavaScript. It provides information about hospital services, doctors, departments, appointments, and contact details through an attractive and user-friendly interface.

## Features
- Responsive Design
- Home Page
- About Hospital
- Departments
- Doctors Information
- Appointment Form
- Contact Section
- Modern UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
Hospital/
│── index.html
│── style.css
│── script.js
```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using VS Code Live Server.
